  
NAME
    logout - Exit a login shell.

SYNOPSIS
    logout [n]

DESCRIPTION
    Exit a login shell.
    
    Exits a login shell with exit status N.  Returns an error if not executed
    in a login shell.

SEE ALSO
    bash(1)
