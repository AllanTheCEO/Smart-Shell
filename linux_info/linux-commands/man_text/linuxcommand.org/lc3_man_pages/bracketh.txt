  
NAME
    [ - Evaluate conditional expression.

SYNOPSIS
    [ arg... ]

DESCRIPTION
    Evaluate conditional expression.
    
    This is a synonym for the "test" builtin, but the last argument must
    be a literal `]', to match the opening `['.

SEE ALSO
    bash(1)
