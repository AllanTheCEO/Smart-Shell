NAME
    readarray - Read lines from a file into an array variable.

SYNOPSIS
    readarray [-d delim] [-n count] [-O origin] [-s count] [-t] [-u fd] [-C callback] [-c quantum] [array]

DESCRIPTION
    Read lines from a file into an array variable.
    
    A synonym for `mapfile'.

SEE ALSO
    bash(1)
