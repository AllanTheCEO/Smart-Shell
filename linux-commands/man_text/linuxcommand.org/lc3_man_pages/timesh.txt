  
NAME
    times - Display process times.

SYNOPSIS
    times

DESCRIPTION
    Display process times.
    
    Prints the accumulated user and system times for the shell and all of its
    child processes.
    
    Exit Status:
    Always succeeds.

SEE ALSO
    bash(1)
