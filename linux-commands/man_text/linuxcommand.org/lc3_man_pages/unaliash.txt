  
NAME
    unalias - Remove each NAME from the list of defined aliases.

SYNOPSIS
    unalias [-a] name [name ...]

DESCRIPTION
    Remove each NAME from the list of defined aliases.
    
    Options:
      -a	remove all alias definitions
    
    Return success unless a NAME is not an existing alias.

SEE ALSO
    bash(1)
