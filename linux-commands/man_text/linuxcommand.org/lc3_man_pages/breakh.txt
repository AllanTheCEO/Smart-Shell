  
NAME
    break - Exit for, while, or until loops.

SYNOPSIS
    break [n]

DESCRIPTION
    Exit for, while, or until loops.
    
    Exit a FOR, WHILE or UNTIL loop.  If N is specified, break N enclosing
    loops.
    
    Exit Status:
    The exit status is 0 unless N is not greater than or equal to 1.

SEE ALSO
    bash(1)
