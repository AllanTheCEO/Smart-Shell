  
NAME
    { ... } - Group commands as a unit.

SYNOPSIS
    { COMMANDS ; }

DESCRIPTION
    Group commands as a unit.
    
    Run a set of commands in a group.  This is one way to redirect an
    entire set of commands.
    
    Exit Status:
    Returns the status of the last command executed.

SEE ALSO
    bash(1)
